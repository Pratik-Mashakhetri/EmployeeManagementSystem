# 🧑‍💼 Employee Management System

A full-stack Employee Management System to manage employee records, including adding, viewing, updating, and deleting employees.

---

## 🚀 Features
- Add new employees  
- View all employees  
- Update employee details  
- Delete employees
- Manage Leaves
- MySQL database integration  

---

## 🛠️ Tech Stack
- Frontend: React (Node.js, npm)  
- Backend: Java (Spring Boot)  
- Database: MySQL
